## 1. What is the difference between Git and GitHub?

Git is a version control system that tracks changes in my project and saves different versions locally. GitHub is an online platform that hosts Git repositories and allows developers to share and collaborate on projects.

## 2. What is the purpose of the terminal?

The terminal is a command-line interface that allows users to communicate directly with the operating system by typing commands instead of using a graphical interface.

## 3. What does git add do?

The git add command moves files from the working directory to the staging area so they are ready to be committed.

## 4. What is the purpose of git commit?

The git commit command saves the staged changes as a permanent snapshot in the local Git repository.
